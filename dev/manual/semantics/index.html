Returns the LogicalOperator that is the dual of the given boolean operator.

Examples

PAndQ.interpret — Function
interpret(valuation, ::Proposition)

Replaces each Atom p in the given Proposition with valuation(p), then simplifies.

Calling p with an incomplete mapping will partially interpret it.

See also tautology and contradiction.

Examples

julia> @p interpret(a -> ⊤, ¬p)
contradiction (generic function with 1 method)

julia> @p interpret(a -> get(Dict(p => ⊤), a, a), p ∧ q)
(q)

Predicates

Base.:== — Function
==(::Union{NullaryOperator, Proposition}, ::Union{NullaryOperator, Proposition})
p == q

Returns a boolean indicating whether p and q are logically equivalent.

Atoms are equivalent if and only if their statements are equivalent.

Info

The ≡ symbol is sometimes used to represent logical equivalence. However, Julia uses ≡ as an alias for the builtin function === which cannot have methods added to it. Use == and === to test for equivalence and identity, respectively.

See also NullaryOperator and Proposition.

Examples

julia> @p p == ¬p
false

julia> @p ¬(p ⊻ q) == (p → q) ∧ (p ← q)
true

julia> @p ¬(p ⊻ q) === (p → q) ∧ (p ← q)
false
