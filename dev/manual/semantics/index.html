PAQ.interpret — Function
interpret(p, valuation...)

Assign a truth value value to p.

Let p be a Proposition. Let valuation be a function, callable object, dictionary, or any number of Pairs that map from atomic propositions in p to their respective truth values.

Calling p with an incomplete mapping will partially interpret p. This returns a Proposition of the same type as p?*? that is independent from every Atoms in valuation.

Base.:== — Function
p == q
==(p, q)
isequal(p, q)

Returns a boolean indicating whether p and q are logically equivalent.

Info

The ≡ symbol is sometimes used to represent logical equivalence. However, Julia uses ≡ as an alias for the builtin function === which cannot have methods added to it. Use == and === to test for equivalence and identity, respectively.

See also Proposition.

Examples

julia> @p p == ¬p
false

julia> @p (p → q) ∧ (p ← q) == ¬(p ⊻ q)
true

julia> @p (p → q) ∧ (p ← q) === ¬(p ⊻ q)
false

PAQ.converse — Function
converse(::BooleanOperator)

Returns the BooleanOperator that is the converse of the given boolean operator.

Examples

julia> converse(and)
and (generic function with 20 methods)

julia> @p and(p, q) == converse(and)(q, p)
true

julia> converse(imply)

julia> @p imply(p, q) == converse(imply)(q, p)
true
