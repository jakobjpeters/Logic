dual(::LogicalOperator)

Returns the LogicalOperator that is the dual of the given boolean operator.

Examples

julia> dual(and)
or (generic function with 18 methods)

julia> @atomize and(p, q) == not(dual(and)(not(p), not(q)))
true

julia> dual(imply)

julia> @atomize imply(p, q) == not(dual(imply)(not(p), not(q)))

PAndQ.interpret — Function
interpret(valuation, ::Proposition)

Substitute each Atom in the given Proposition with values from the valuation.

The valuation can be a function or function-like object with the signature valuation(::Atom)::Union{Bool, NullaryOperator}, a Dict, or an iterable that can construct a Dict. No substitution is performed if an Atom from the Proposition is not one of the dictionary's keys.

See also NullaryOperator.

Examples

julia> @atomize interpret(a -> true, ¬p)
false

julia> @atomize interpret(p => true, p ∧ q)
q
source

Base.:== — Function
==(::Union{NullaryOperator, Proposition}, ::Union{NullaryOperator, Proposition})
p == q

Returns a boolean indicating whether p and q are logically equivalent.

Constants are equivalent if and only if their values are equivalent.

Info

The ≡ symbol is sometimes used to represent logical equivalence. However, Julia uses ≡ as an alias for the builtin function === which cannot have methods added to it. Use == and === to test for equivalence and identity, respectively.

See also NullaryOperator and Proposition.

Examples

julia> @atomize p == ¬p
false

julia> @atomize ¬(p ⊻ q) == (p → q) ∧ (p ← q)
true

julia> @atomize ¬(p ⊻ q) === (p → q) ∧ (p ← q)
false
