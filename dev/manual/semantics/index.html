Semantics

Base.:== — Function
p == q
==(p, q)

Returns a boolean indicating whether p and q are logically equivalent.

Info

The ≡ symbol is sometimes used to represent logical equivalence. However, Julia uses ≡ as an alias for the builtin function === which cannot have methods added to it. Use == and === to test for equivalence and identity, respectively.

See also Proposition.

Examples

julia> @p p == ¬p
false

julia> @p ¬(p ⊻ q) == (p → q) ∧ (p ← q)
true

julia> @p ¬(p ⊻ q) === (p → q) ∧ (p ← q)
false

PAQ.interpret — Function
interpret(p::Union{NullaryOperator, Proposition}, valuation...)

Replaces each Atom in p with its truth value in valuation, then simplifies.

valuation is either a Dict or a set that can construct one that maps from atoms to their respective truth values.

Calling p with an incomplete mapping will partially interpret it.

See also tautology and [contradiction].

Examples

julia> @p interpret(¬p, p => ⊤)
contradiction (generic function with 1 method)

julia> @p p = Clause(and, q, r, s)
Clause:
 q ∧ r ∧ s

julia> @p interpret(p, q => ⊤, r => ⊤)
Clause:
 s

PAQ.valuations — Function
valuations(atoms)
valuations(::Proposition)

Return an iterator containing every possible valuation of the Atoms.

A valuation is a vector of Pairs which map from an atom to a truth value.

Examples

julia> @p collect(valuations([p]))
2-element Vector{Vector}:
 Pair{Atom{Symbol}, typeof(tautology)}[p => PAQ.tautology]
 Pair{Atom{Symbol}, typeof(contradiction)}[p => PAQ.contradiction]

julia> @p collect(valuations([p, q]))
4-element Vector{Vector}:
 Pair{Atom{Symbol}, typeof(tautology)}[p => PAQ.tautology, q => PAQ.tautology]
 Pair{Atom{Symbol}}[p => PAQ.contradiction, q => PAQ.tautology]
 Pair{Atom{Symbol}}[p => PAQ.tautology, q => PAQ.contradiction]
 Pair{Atom{Symbol}, typeof(contradiction)}[p => PAQ.contradiction, q => PAQ.contradiction]

PAQ.solve — Function
solve(p, truth_value = ⊤)

Return a vector containing all interpretations such that interpret(p, interpretation) == truth_value.

Examples

julia> @p collect(solve(p))
1-element Vector{Vector{Pair{Atom{Symbol}, typeof(tautology)}}}:
 [p => PAQ.tautology]

julia> @p collect(solve(p ⊻ q, ⊥))
2-element Vector{Vector}:
 Pair{Atom{Symbol}, typeof(tautology)}[p => PAQ.tautology, q => PAQ.tautology]
 Pair{Atom{Symbol}, typeof(contradiction)}[p => PAQ.contradiction, q => PAQ.contradiction]
