PAQ.Primitive — Type
Primitive <: Language

Primitive proposition.

Subtype of Language. See also Compound.

Examples

julia> p
Primitive("p")

julia> p()

PAQ.Propositional — Type
Propositional{
    L <: Union{
        Primitive,
        Tuple{Not, Compound},
        Tuple{And, Compound, Compound}
    }
}(ϕ::L) <: Compound <: Language

Abstract syntax tree representing a compound proposition.

Subtype of Compound and Language. See also Primitive, Not, and And.

Examples

julia> p ∧ ¬p
Propositional(
  And(), Propositional(
    Primitive("p")
  ) Propositional(
    Not(), Propositional(
      Primitive("p")
    ) 
  ) 
)

julia> (p ∧ ¬p)()
2-element Vector{Pair{Vector{Pair{Primitive, Truth}}, Truth{Val{:⊥}}}}:
 [Primitive("p") => ⊤] => ⊥
 [Primitive("p") => ⊥] => ⊥

julia> (p → q) ∧ (p ← q) == ¬(p ⊻ q)
true
source
PAQ.Truth — Type
Truth{V <: Union{Val{:⊥}, Val{:⊤}}}(::V) <: Language

Container for the constants tautology and contradiction. Subtype of Language.

source
PAQ.tautology — Constant
⊤
tautology

A constant which is true in every possible interpretation.

One of two valid instances of Truth, the other instance being contradiction.

⊤ can be typed by \top<tab>.

Examples

julia> ¬⊤
⊥

julia> tautology()
1-element Vector{Pair{Vector{Pair{Primitive, Truth}}, Truth{Val{:⊤}}}}:
 [] => ⊤
source
PAQ.contradiction — Constant
⊥
contradiction

A constant which is false in every possible interpretation.

One of two valid instances of Truth, the other instance being tautology.

⊥ can be typed by \bot<tab>.

Examples

julia> ¬⊥
⊤

julia> contradiction()
1-element Vector{Pair{Vector{Pair{Primitive, Truth}}, Truth{Val{:⊥}}}}:
 [] => ⊥

PAQ.get_primitives — Function
get_primitives(ps::Language...)

Returns a vector of Primitive propositions contained in p.

Note that some primitives may optimized out of an expression, such as in p ∧ ⊥.

See also Language.

Examples

julia> get_primitives(p)
1-element Vector{Primitive}:
 Primitive("p")

julia> get_primitives(p ∧ q, r)
3-element Vector{Primitive}:
 Primitive("p")
 Primitive("q")
 Primitive("r")
