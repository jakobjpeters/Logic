PAQ.Propositional — Type
Propositional{
    L <: Union{
        Tuple{Not, Compound},
        Tuple{And, Compound, Compound}
    }
} <: Compound <: Language
Propositional(ϕ::L)

Abstract syntax tree representing a compound proposition.

Subtype of Compound and Language. See also Not and And.

Examples

julia> r = p ∧ ¬p
Propositional:
  p ∧ ¬p

julia> r()
Truth:
  ⊥

julia> (p ∧ q)()
Contingency:
  [p => ⊤, q => ⊤] => ⊤
  [p => ⊤, q => ⊥] => ⊥
  [p => ⊥, q => ⊤] => ⊥
  [p => ⊥, q => ⊥] => ⊥

PAQ.Normal — Type
Normal{B <: Union{And, Or}} <: Compound <: Language
Normal(::B, clauses::Vector{Vector{Literal}})

The conjunctive or disjunctive normal form of a proposition.

Constructing an instance with the parameters (And(), p) and (Or(), p) correspond to conjunctive and disjunctive normal form, respectively.

Subtype of Compound and Language.

Examples

julia> r = Normal(And(), p ∧ q)
Normal:
  (¬p ∨ q) ∧ (p ∨ ¬q) ∧ (p ∨ q)

julia> s = Normal(Or(), ¬p ∨ ¬q)
Normal:
  (p ∧ ¬q) ∨ (¬p ∧ q) ∨ (¬p ∧ ¬q)

julia> t = r ∧ s
Propositional:
  (¬p ∨ q) ∧ (p ∨ ¬q) ∧ (p ∨ q) ∧ (p ∧ ¬q) ∨ (¬p ∧ q) ∨ (¬p ∧ ¬q)

julia> t()
Truth:
  ⊥

PAQ.tautology — Constant
⊤
tautology

A constant which is true in every possible interpretation.

One of two valid instances of Truth, the other instance being contradiction.

⊤ can be typed by \top<tab>.

Examples

julia> ¬⊤

  ⊥

  ⊤

PAQ.contradiction — Constant

PAQ.get_primitives — Function
get_primitives(ps::Language...)

Returns a vector of Primitive propositions contained in ps.

Note that some primitives may optimized out of an expression, such as in p ∧ ⊥.

See also Language.

Examples

julia> r = p ∧ q
Propositional:
  p ∧ q

julia> get_primitives(r)
2-element Vector{Primitive}:
 Primitive("p")
 Primitive("q")
