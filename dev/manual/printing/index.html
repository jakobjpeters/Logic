PAndQ.TruthTable — Type
TruthTable(::Vector{String}, ::Matrix{Bool})
TruthTable(ps)

Construct a truth table for the given Propositions.

The header is a vector containing vectors of logically equivalent propositions. The body is a matrix where the rows contain interpretations of each proposition in the given column.

See also Nullary Operators.

Examples

julia> TruthTable([⊤])
┌───┐
│ ⊤ │
├───┤
│ ⊤ │
└───┘

julia> @atomize TruthTable([¬p])
┌───┬────┐
│ p │ ¬p │
├───┼────┤
│ ⊤ │ ⊥  │
│ ⊥ │ ⊤  │
└───┴────┘

julia> @atomize TruthTable([p ∧ ¬p, p ⊻ q, ¬(p ∧ q) ∧ (p ∨ q)])
┌────────┬───┬───┬───────────────────────────┐
│ p ∧ ¬p │ p │ q │ p ⊻ q, ¬(p ∧ q) ∧ (p ∨ q) │
├────────┼───┼───┼───────────────────────────┤
│ ⊥      │ ⊤ │ ⊤ │ ⊥                         │
│ ⊥      │ ⊥ │ ⊤ │ ⊤                         │
├────────┼───┼───┼───────────────────────────┤
│ ⊥      │ ⊤ │ ⊥ │ ⊤                         │
│ ⊥      │ ⊥ │ ⊥ │ ⊥                         │
└────────┴───┴───┴───────────────────────────┘

Base.show — Function
show(::IO, ::MIME"text/plain", ::Proposition)

Represent the given Proposition as a propositional formula.

The value of a Constant is shown with IOContext(io, :compact => get(io, :compact, true)).

Examples

julia> @atomize show(stdout, MIME"text/plain"(), p ⊻ q)
p ⊻ q

julia> @atomize show(stdout, MIME"text/plain"(), PAndQ.Normal(p ⊻ q))
(p ∨ q) ∧ (¬p ∨ ¬q)

AbstractTrees.print_tree — Function
print_tree(::IO = stdout, ::Proposition; kwargs...)

Prints a tree diagram of the given Proposition.

See also AbstractTrees.print_tree.

julia> @atomize print_tree(p ∧ ¬q ⊻ s)
⊻
├─ ∧
│  ├─ 𝒾
│  │  └─ p
│  └─ ¬
│     └─ q
└─ 𝒾
   └─ s

∧
├─ ∨
│  ├─ 𝒾
│  │  └─ p
│  └─ 𝒾
│     └─ s
├─ ∨
│  ├─ ¬
│  │  └─ q
│  └─ 𝒾
│     └─ s
└─ ∨
   ├─ ¬
   │  └─ p
   ├─ 𝒾

      └─ s
