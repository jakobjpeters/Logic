AbstractTrees.jl

AbstractTrees.children — Function
children(::Proposition)

Return an iterator over the leaf nodes of the given Proposition.

Warning

Nodes that apply the identity function are skipped.

Examples

julia> @p PAndQ.children(Literal(identity, p))
()

julia> @p PAndQ.children(Literal(not, p))
(Atom(:p),)

julia> @p PAndQ.children(p ⊻ q)
2-element Vector{Tree{typeof(identity), Atom{Symbol}}}:
 p
 q

julia> @p PAndQ.children(Normal(and, p ⊻ q))
2-element Vector{Clause{typeof(or)}}:
 p ∨ q
 ¬p ∨ ¬q

PAndQ.atomize — Function
atomize(x)

If x is a symbol, return an expression that instantiates it as an Atom if it is undefined. If x is an expression, traverse it with recursive calls to atomize (ignoring variable assignment and keyword arguments). Otherise, return x.
