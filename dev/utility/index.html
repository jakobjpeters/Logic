PAQ.@primitive — Macro
@primitive

Instantiates Primitive propositions.

Examples

julia> @primitive p q

julia> p
Primitive("p")

julia> q
Primitive("q")

PAQ.truth_table — Function
truth_table(operator)

Return a vector of every possible input => output pair where the inputs are Valuations.

julia> truth_table(¬)
2-element Vector{Pair}:
 ⊤ => ⊥
 ⊥ => ⊤

julia> truth_table(∧)
4-element Vector{Pair}:
 (⊤, ⊥) => ⊥
 (⊥, ⊥) => ⊥
 (⊤, ⊤) => ⊤
 (⊥, ⊤) => ⊥
